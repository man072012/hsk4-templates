# HSK4 — قوالب الكتابة

نسخة مُعاد تنظيمها من [hsk4-writing](https://man072012.github.io/hsk4-writing/) — بدلاً من التصنيف حسب التدريبات، يُنظَّم المحتوى:

- **القسم الأول (ترتيب الكلمات 86–95):** حسب القاعدة النحوية. 21 قاعدة، تحت كل قاعدة جميع الأسئلة التي تستخدمها.
- **القسم الثاني (الصور 96–100):** حسب قالب التعبير. 11 قالباً موحَّداً يغطي كل الكلمات الـ52 من بنك الامتحان، مع جملة مثال جاهزة لكل قالب — بدّل الكلمة فقط، تحصل على جملة صحيحة.
